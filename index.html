Welcome to String Decorator!

このプログラムは、Minecraft Java Editionで装飾文字を入力するための文字変換を提供するものです。

提供している機能は以下の通りです。

  • 文字への装飾追加、削除
  • ASCIIコードへのエンコード、デコード
  • 装飾記号の切り替え
  • 変換後の文字列をクリップボードにコピー
